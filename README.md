# لوحة إدخال مؤشرات الجودة للمراكز الصحية HQI 2026 - نسخة سريعة

هذه نسخة مستقلة وبسيطة مخصصة فقط لمؤشرات المراكز الصحية الموجودة في ملف `EJH - HQI 2026.xlsx`.

عدد المؤشرات المفعلة في هذه النسخة: 18 مؤشر.

## التشغيل المحلي
```bash
cd hqi-quality-quickstart
npm install
npm run dev
```

## ربط Supabase قبل إرسال الرابط للمراكز
1. أنشئ مشروع Supabase.
2. افتح SQL Editor.
3. شغّل الملف:
```bash
database/supabase_setup.sql
```
4. انسخ `.env.example` إلى `.env.local` وضع مفاتيح الاتصال.

## النشر على Vercel
ارفع المشروع إلى GitHub ثم اربطه في Vercel، وأضف متغيرات البيئة نفسها في إعدادات المشروع.

## المراكز الموجودة
الواحة، الرغامة، قويزة، المطار القديم، الروابي، السليمانية، شرق الخط.
